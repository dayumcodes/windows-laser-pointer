# Assets Directory

This directory should contain the following image assets for the Windows Store package:

- `SplashScreen.scale-200.png` - 620x300 pixels
- `LockScreenLogo.scale-200.png` - 48x48 pixels
- `Square150x150Logo.scale-200.png` - 300x300 pixels
- `Square44x44Logo.scale-200.png` - 88x88 pixels
- `Square44x44Logo.targetsize-24_altform-unplated.png` - 24x24 pixels
- `StoreLogo.png` - 50x50 pixels
- `Wide310x150Logo.scale-200.png` - 620x300 pixels

These assets are required for Windows Store submission. You can create placeholder images or use a tool to generate them.

For now, the project will build without these assets, but they should be added before Store submission.
